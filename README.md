# DH101 Student Website

Starter repository for a college student website in DH 101 (Spring 2026).

## What this site includes

- A responsive homepage in `index.html`
- Weekly work logs in `makes/`
- Weekly reflections in `reflections/`
- Supporting pages in `pages/`

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. Open your repository on GitHub.
3. Go to **Settings** -> **Pages**.
4. In **Build and deployment**, set:
	- **Source**: Deploy from a branch
	- **Branch**: `main` (or `master`), folder `/ (root)`
5. Save, then wait about 1-3 minutes.
6. Open the published URL shown in the Pages settings.

## Editing content

- Update homepage text and sections in `index.html`.
- Add new entries in `makes/` and `reflections/` each week.
- Edit informational pages in `pages/`.

Markdown help: see `pages/markdown-guide.md`.
